# space_shoter
🚀 Penjelasan Game Space Warship

Space Warship adalah game arcade bertema luar angkasa yang terinspirasi dari game klasik Space Invaders. Dalam game ini, pemain mengendalikan sebuah pesawat luar angkasa yang bertugas mempertahankan bumi dari serangan alien.

Pemain harus menembakkan peluru ke arah musuh yang tersusun dalam beberapa baris di bagian atas layar. Setiap alien yang berhasil dihancurkan akan menambah skor pemain. Semakin banyak musuh yang dikalahkan, semakin tinggi skor yang diperoleh.

🎮 Cara Bermain

Pemain mengontrol pesawat luar angkasa di bagian bawah layar.

Pesawat dapat bergerak ke kiri dan ke kanan.

Pemain menembakkan peluru ke arah musuh alien.

Musuh akan berkurang jika terkena peluru.

Game berakhir jika musuh mencapai area bawah atau pemain terkena serangan.

⭐ Fitur Utama Game

Pesawat Pemain: Digunakan untuk menyerang musuh.

Musuh Alien: Target utama yang harus dihancurkan.

Skor (Score): Menampilkan jumlah musuh yang berhasil dikalahkan.

Latar Luar Angkasa: Memberikan nuansa visual galaksi dengan planet dan bintang.

Gameplay Sederhana & Menantang: Mudah dimainkan tetapi membutuhkan fokus dan refleks cepat.

🎯 Tujuan Game

Tujuan utama dari game Space Warship adalah:

Mengalahkan semua musuh alien

Mendapatkan skor setinggi mungkin

Melatih konsentrasi dan kecepatan reaksi pemain

🧠 Manfaat Game

Melatih koordinasi tangan dan mata

Mengasah kemampuan logika dan strategi

Memberikan hiburan dengan konsep klasik yang seru
